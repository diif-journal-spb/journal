В работе рассматривается система автоматического управления вида 
\begin{equation}
\left\{ {\begin{array}{l}
 \dot {x}=y \\ 
 \dot {y}=-\alpha y-\beta x-\varphi \,\left[ {t,\sigma ,\varphi _0 } 
\right],\; \\ 
 \end{array}} \right.
\end{equation}
где $\sigma =ay+bx$. Предполагается, что $\alpha >0,\quad \beta >0$ и 
$b^2-\alpha ab+a^2\beta \ne 0$, т.е. при $\varphi \,\left[ {t,\sigma 
,\varphi _0 } \right]\equiv 0$ система (1) является асимптотически 
устойчивой и передаточная функция системы является невырожденной. 

$\varphi \,\left[ {t,\sigma ,\varphi _0 } \right]\quad -$ 
релейно-гистерезисная функция, состоящая из двух ветвей однозначных функций 
$\varphi _1 \left( {t,\sigma } \right)$ и $\varphi _2 \left( {t,\sigma } 
\right)$, изображенная на рис. 1.

Строгое общее определение гистерезисных нелинейностей введено в работах В.А. 
Якубовича [12], М.А. Красносельского и А.В. Покровского [7]. Значение 
функции $\varphi \,\left[ {t,\sigma ,\varphi _0 } \right]$ зависит от $t$, 
функции $\sigma \left( t \right)$ и от начального значения $\varphi _0 $.

В нашем случае гистерезисная функция $\varphi \,\left[ {t,\sigma ,\varphi _0 
} \right]$ состоит из двух ветвей однозначных функций $\varphi _1 \left( 
{t,\sigma } \right)$ и $\varphi _2 \left( {t,\sigma } \right)$:
\begin{equation}
\left\{ {\begin{array}{l}
 \varphi \,\left[ {t,\sigma ,\varphi _0 } \right]=\varphi _{\,1} \left( 
{t,\,\sigma } \right),\;\;\mbox{если}\;\;\sigma \ge -\delta , \\ 
 \varphi \,\left[ {t,\sigma ,\varphi _0 } \right]=\varphi _{\,2} \left( 
{t,\,\sigma } \right),\;\;\mbox{если}\;\;\sigma \le \delta \;\;\left( 
{\delta >0} \right), \\ 
 \end{array}} \right.
\end{equation}
направление обхода петли гистерезиса указано на рис. 1 стрелками.

Если $\sigma _0 >\delta $, то $\varphi _0 =\varphi _1 \left( {0,\sigma _0 } 
\right)$; если $\sigma _0 <-\delta $, то $\varphi _0 =\varphi _2 \left( 
{0,\sigma _0 } \right)$; в случае $-\delta \le \sigma _0 \le \delta $ 
множество начальных значений состоит из двух точек: $\varphi _0 \in \left\{ 
{\varphi _1 \left( {0,\sigma _0 } \right),\,\,\varphi _2 \left( {0,\sigma _0 
} \right)} \right\}$. $S\le \varphi _1 \left( {t,\sigma } \right)\le M,
\quad -M\le \varphi _2 \left( {t,\sigma } \right)\le -S,$ где $S,\;M$ - некоторые 
константы, $0<S<M$.

Двумерные системы вида (1) с различными нелинейностями $\varphi \,\left[ 
{t,\sigma ,\varphi _0 } \right]$ возникают во многих прикладных задачах 
теории управления и достаточно хорошо изучены, их исследованию посвящено 
большое количество работ и монографий (например, [1-4, 6, 10-11]). Одной из 
основных задач является определение значений параметров системы, при которых 
положения равновесия или стационарные множества системы являются 
устойчивыми.

Понятие абсолютной устойчивости систем автоматического управления с 
нелинейностью $\varphi \left( {t,\sigma } \right)$, удовлетворяющей 
секторному условию ${0\le \varphi \left( {t,\sigma } \right)\sigma \le 
k\sigma ^2}$ для $\forall t\in {\rm R},\;\forall \sigma \in {\rm R}$, было 
впервые введено в работе [9]. Исследование устойчивости таких систем 
проводилось с помощью построения функций Ляпунова или с помощью частотных 
методов (например, [6, 9]), которые позволяют получить лишь достаточные 
условия абсолютной устойчивости. Г.А. Леоновым в работе [8] с помощью метода 
систем сравнения получены эффективные и легко проверяемые необходимые и 
достаточные условия абсолютной устойчивости таких систем.

Системы с неединственным состоянием равновесия и с нелинейностью, не 
удовлетворяющей секторному условию, также достаточно широко изучены 
(например, [4, 6, 13]). Большое число работ посвящено исследованию систем с 
различными релейными нелинейностями (например, [11, 13]), и в частности, с 
релейно-гистерезисной нелинейностью вида $\varphi \,\left[ {t,\sigma 
,\varphi _0 } \right]$, где 
\begin{equation}
\left\{ {\begin{array}{l}
 \varphi \,\left[ {t,\sigma ,\varphi _0 } 
\right]=M,\;\;\mbox{если}\;\;\sigma \ge -\delta , \\ 
 \varphi \,\left[ {t,\sigma ,\varphi _0 } 
\right]=-M,\;\;\mbox{если}\;\;\sigma \le \delta \;\;\left( {M,\,\,\delta >0} 
\right), \\ 
 \end{array}} \right.
\end{equation}
которая является частным случаем нелинейности (2).

Понятие абсолютной устойчивости для систем вида (1) с нелинейностью (2) 
строго не сформулировано, хотя и используется в литературе (например, [2, 
3]).

В данной работе вводится определение абсолютной устойчивости двумерных 
систем вида (1) с релейно-гистерезисной нелинейностью (2) и через 
коэффициенты системы даются необходимые и достаточные условия абсолютной 
устойчивости таких систем. Эти условия получены с помощью метода систем 
сравнения. В качестве систем сравнения рассматриваются системы вида (1) с 
нелинейностью вида (3). Необходимые и достаточные условия устойчивости в 
целом стационарного множества таких систем приведены в [5]. Результаты 
работы являются новыми и могут быть использованы для решения ряда 
теоретических задач и более детального изучения реальных релейных систем, 
где кусочно-линейные модели становятся слишком упрощенными и требуется учет 
различных внешних воздействий.

Фазовая поверхность системы (1) с нелинейностью вида (2) состоит из двух 
листов: $P_{\,1} =\left\{ {\left( {x,y} \right):\;\sigma \ge -\delta } 
\right\}$ и $P_{\,2} =\left\{ {\left( {x,y} \right):\;\sigma \le \delta } 
\right\}$, перекрывающих друг друга в ``зоне неоднозначности'' $-\delta \le 
\sigma \le \delta $. 

Переход фазовой точки с первого листа на второй происходит по лучу $L_{\,1} 
=\left\{ {\left( {x,y} \right):\sigma =-\delta ,\;\dot {\sigma }\left| 
{_{\sigma \to -\delta +0} } \right.\le 0} \right\}$, переход со второго 
листа на первый - по лучу $L_{\,2} =\left\{ {\left( {x,y} \right):\sigma 
=\delta ,\;\dot {\sigma }\left| {_{\sigma \to \delta -0} } \right.\ge 0} 
\right\}$ (рис. 1). 

Решением системы (1) с нелинейностью (2) на листе $P_{\,1} $ является 
решение системы 
\begin{equation}
\left\{ {\begin{array}{l}
 \dot {x}=y \\ 
 \dot {y}=-\alpha y-\beta x-\varphi _{\,1} \left( {t,\,\sigma } \right),\; 
\\ 
 \end{array}} \right.
\end{equation}
на листе $P_{\,2} $ - решение системы
\begin{equation}
\left\{ {\begin{array}{l}
 \dot {x}=y \\ 
 \dot {y}=-\alpha y-\beta x-\varphi _{\,2} \left( {t,\,\sigma } \right).\; 
\\ 
 \end{array}} \right.
\end{equation}
Предполагаем, что функции $\varphi _{\,i} \left( {t,\,\sigma } \right):{\rm 
R}_+ \times {\rm R}\to {\rm R}$ ($i=1,\,\,2)$ таковы, что на листе $P_{\,1} 
$ выполнены условия существования и единственности решения системы (4), на 
листе $P_{\,2} $ - условия существования и единственности решения системы 
(5).

Решение системы (4) на листе $P_{\,1} $, достигающее в момент времени 
$t=\tau _1 $ луча $L_{\,1} $ в некоторой точке $\left( {x_1 ,\,y_1 } 
\right)$, продолжается при $t>\tau _1 $ на лист $P_{\,2} $ и является 
решением системы (5) с начальными данными $\left( {\tau _1 ,\;x_1 ,\,y_1 } 
\right)$. 

Аналогично, решение (5) на листе $P_{\,2} $, достигающее в момент $t=\tau _2 
$ луча $L_{\,2} $ в точке $\left( {x_2 ,\,y_2 } \right)$, продолжается при 
$t>\tau _2 $ на лист $P_{\,1} $ и является решением системы (4) с начальными 
условиями $\left( {\tau _2 ,\,x_2 ,\,y_2 } \right)$.

\textit{Определение 1}. Будем говорить, что гистерезисная функция $\varphi \,\left[ {t,\sigma 
,\varphi _0 } \right]$, удовлетворяющая описанным выше условиям, является \textit{функцией класса } $K_{S,M} $.

Не умаляя общности рассуждений можно считать, что $a\ge 0$. Для 
доказательства этого факта в системе достаточно сделать замену $x\to 
-x,\;\,y\to -y$ и заметить, что функция $-\varphi \,\left[ {t,\sigma 
,\varphi _0 } \right]$ тоже принадлежит классу $K_{S,M} $.

Рассмотрим случай $a>0$ (случай $a=0$ рассматривается аналогично).

Введем в рассмотрение два отрезка: 

$J_1 =\left\{ {\left( {x,y} \right):\;-M \mathord{\left/ {\vphantom {M \beta 
}} \right. \kern-\nulldelimiterspace} \beta \le x\le -S \mathord{\left/ 
{\vphantom {S \beta }} \right. \kern-\nulldelimiterspace} \beta ,\;y=0} 
\right\}$ и

 $J_2 =\left\{ {\left( {x,y} \right):\;S \mathord{\left/ 
{\vphantom {S \beta }} \right. \kern-\nulldelimiterspace} \beta \le x\le M 
\mathord{\left/ {\vphantom {M \beta }} \right. \kern-\nulldelimiterspace} 
\beta ,\;y=0} \right\}$.

Заметим, что если при $x=x_0 ,\;\,y=y_0 $ и некотором $t=t_0 $ правые части 
системы (4) обращаются в ноль, то точка $\left( {x_0 ,\;y_0 } \right)$ 
принадлежит множеству $J_1 $. И каждая точка отрезка $J_1 $ является особой 
точкой системы (4) при $\varphi _{\,1} \left( {t,\,\sigma } 
\right)=c=const$, где $S\le c\le M$.

Аналогично, если при $x=x_0 ,\;\,y=y_0 $ и некотором $t=t_0 $ правые части 
системы (5) обращаются в ноль, то $\left( {x_0 ,\;y_0 } \right)\in J_2 $. И 
каждая точка $J_2 $ является особой точкой системы (5) при $\varphi _{\,2} 
\left( {t,\,\sigma } \right)=c=const$, где $-M\le c\le -S$.

Если выполнено неравенство $\delta \beta -Mb\ge 0$, то отрезок $J_1 $ 
целиком лежит в $P_{\,1} $, а отрезок $J_2 $ целиком лежит в $P_{\,2} $.

\textit{Определение 2.} Множество $\Theta =J_1 \cup J_2 $ назовем \textit{сингулярным множеством} для системы (1) с функцией 
$\varphi \,\left[ {t,\sigma ,\varphi _0 } \right]$ класса $K_{S,M} $.

\textit{Определение 3.} Множество $\Theta $ называется \textit{устойчивым множеством }системы (1) с функцией $\varphi \,\left[ 
{t,\sigma ,\varphi _0 } \right]$ класса $K_{S,M} $, если для любого 
$\varepsilon >0$ найдется такое $\delta >0$, что для каждого решения $\left( 
{x\left( t \right),\;y\left( t \right)} \right)$ системы с начальными 
условиями $\left( {t_0 ,\;x_0 ,\;y_0 } \right)$ выполнение неравенства $\rho 
\left( {\left( {x\left( {t_0 } \right),\;y\left( {t_0 } \right)} 
\right),\;\Theta } \right)<\delta $ влечет за собой выполнение неравенства 
$\rho \left( {\left( {x\left( t \right),\;y\left( t \right)} 
\right),\;\Theta } \right)<\varepsilon $ для всех $t>t_0 $.

\textit{Определение 4.} Множество $\Theta $ называется \textit{устойчивым в целом множеством }системы (1) с функцией $\varphi \,\left[ 
{t,\sigma ,\varphi _0 } \right]$ класса $K_{S,M} $, если оно устойчиво и для 
каждого решения $\left( {x\left( t \right),\;y\left( t \right)} \right)$ 
системы $\rho \left( {\left( {x\left( t \right),\;y\left( t \right)} 
\right),\;\Theta } \right)\to 0$ при $t\to +\infty $.

\textit{Определение 5.} Система (1) называется \textit{абсолютно устойчивой} в классе нелинейностей $K_{S,M} $, если для любой 
функции $\varphi \,\left[ {t,\sigma ,\varphi _0 } \right]$ из этого класса 
множество $\Theta $ является для системы (1) устойчивым в целом.

Изучим теперь поведение решений системы (1) с нелинейностью класса $K_{S,M} 
$.

Введем в рассмотрение системы сравнения.

На листе $P_{\,1} $:

при $y>0$
\[
\frac{-\alpha y-\beta x-\left( {M+\varepsilon } 
\right)}{y}<\frac{dy}{dx}=\frac{-\alpha y-\beta x-\varphi _{\,1} \left( 
{t,\sigma } \right)}{y}<\frac{-\alpha y-\beta x-\left( {S-\varepsilon } 
\right)}{y},
\]

при $y<0$
\[
\frac{-\alpha y-\beta x-\left( {S-\varepsilon } 
\right)}{y}<\frac{dy}{dx}=\frac{-\alpha y-\beta x-\varphi _{\,1} \left( 
{t,\sigma } \right)}{y}<\frac{-\alpha y-\beta x-\left( {M+\varepsilon } 
\right)}{y}
\]
для любого сколь угодно малого $\varepsilon >0$. 

Обозначим $M+\varepsilon =\bar {M}$, $S-\varepsilon =\bar {S}$. 

Из неравенств следует, что при $y>0$ решения системы (4) пересекают 
``снаружи вовнутрь'' решения системы 
\begin{equation}
\left\{ {\begin{array}{l}
 \dot {x}=y \\ 
 \dot {y}=-\alpha y-\beta x-\bar {S},\; \\ 
 \end{array}} \right.
\end{equation}
и пересекают ``изнутри наружу'' решения системы
\begin{equation}
\left\{ {\begin{array}{l}
 \dot {x}=y \\ 
 \dot {y}=-\alpha y-\beta x-\bar {M}.\; \\ 
 \end{array}} \right.
\end{equation}
При $y<0$ решения системы (4) пересекают ``снаружи вовнутрь'' решения 
системы (7) и пересекают ``изнутри наружу'' решения системы (6).

На листе $P_{\,2} $:

при $y>0$
\[
\frac{-\alpha y-\beta x+\left( {S-\varepsilon } 
\right)}{y}<\frac{dy}{dx}=\frac{-\alpha y-\beta x-\varphi _{\,2} \left( 
{t,\sigma } \right)}{y}<\frac{-\alpha y-\beta x+\left( {M+\varepsilon } 
\right)}{y},
\]
при $y<0$
\[
\frac{-\alpha y-\beta x+\left( {M+\varepsilon } 
\right)}{y}<\frac{dy}{dx}=\frac{-\alpha y-\beta x-\varphi _{\,2} \left( 
{t,\sigma } \right)}{y}<\frac{-\alpha y-\beta x+\left( {S-\varepsilon } 
\right)}{y}
\]
для любого сколь угодно малого $\varepsilon >0$.

Поэтому при $y>0$ решения системы (5) пересекают ``изнутри наружу'' решения 
системы 
\begin{equation}
\left\{ {\begin{array}{l}
 \dot {x}=y \\ 
 \dot {y}=-\alpha y-\beta x+\bar {S},\; \\ 
 \end{array}} \right.
\end{equation}
и пересекают ``снаружи вовнутрь'' решения системы
\begin{equation}
\left\{ {\begin{array}{l}
 \dot {x}=y \\ 
 \dot {y}=-\alpha y-\beta x+\bar {M}.\; \\ 
 \end{array}} \right.
\end{equation}
При $y<0$ решения (5) пересекают ``снаружи вовнутрь'' решения системы (8) и 
пересекают ``изнутри наружу'' решения системы (9).

Если $b\left( {b^2-\alpha ab+\beta a^2} \right)>0$, то лучи $L_1 $ и $L_2 $ 
определяются следующим образом: $L_1 =\left\{ {\left( {x,y} 
\right):\;\;ay+bx=-\delta ,\,\;y\le -\gamma _1 } \right\}$, $L_2 =\left\{ 
{\left( {x,y} \right):\;\;ay+bx=\delta ,\;\,y\ge \gamma _2 } \right\}$, 
$\gamma _1 =\frac{a\left( {\delta \beta -\varphi _{\,1} \left( {t_0 ,-\delta 
} \right)b} \right)}{b^2-\alpha ab+\beta a^2},
\quad
\gamma _2 =\frac{a\left( {\delta \beta +\varphi _{\,2} \left( {t_0 ,\delta } 
\right)b} \right)}{b^2-\alpha ab+\beta a^2},$ и $\gamma _{\bar {M}} \le 
\gamma _{1,\,2} \le \gamma _{\bar {S}} $ для достаточно малого $\varepsilon 
>0$, где $\gamma _{\bar {M}} =\frac{a\left( {\delta \beta -\bar {M}b} 
\right)}{b^2-\alpha ab+\beta a^2}$, $\gamma _{\bar {S}} =\frac{a\left( 
{\delta \beta -\bar {S}b} \right)}{b^2-\alpha ab+\beta a^2}$.

Если $b\left( {b^2-\alpha ab+\beta a^2} \right)<0$, то $L_1 =\left\{ {\left( 
{x,y} \right):\;\;ay+bx=-\delta ,\;\,y\ge -\gamma _1 } \right\}$, $L_2 
=\left\{ {\left( {x,y} \right):\;\;ay+bx=\delta ,\;\,y\le \gamma _2 } 
\right\}$ и $\gamma _{\bar {S}} \le \gamma _{1,\,2} \le \gamma _{\bar {M}} 
$.

Если $b=0$, то $L_1 =\left\{ {\left( {x,y} \right):\;\;ay=-\delta ,\;\,x\ge 
-\eta _1 } \right\}$, $L_2 =\left\{ {\left( {x,y} \right):\;\;ay=\delta 
,\;\,x\le \eta _2 } \right\}$, где $\eta _1 =\frac{\varphi _{\,1} \left( 
{t_0 ,-\delta } \right)a-\delta \alpha }{\beta a}$, $\eta _2 =\frac{-\varphi 
_{\,2} \left( {t_0 ,\delta } \right)a-\delta \alpha }{\beta a}$, и ${\eta 
_{\bar {S}} \le \eta _{1,\,2} \le \eta _{\bar {M}}}$, где $\eta _{\bar {M}} 
=\frac{\bar {M}a-\delta \alpha }{\beta a}$, $\eta _{\bar {S}} =\frac{\bar 
{S}a-\delta \alpha }{\beta a}$.

\textbf{Теорема 1}.\textit{ Пусть} $\quad b>0,\quad \delta \beta -Mb>0$ \textit{и выполнено одно из следующих условий 1 - 7:}

\textit{1) }$ \alpha ^2-4\beta >0,\;\;\lambda _2 >-b \mathord{\left/ {\vphantom {b a}} 
\right. \kern-\nulldelimiterspace} a;$

\textit{2) }$ \alpha ^2-4\beta >0,\;\;\lambda _2 <-b \mathord{\left/ {\vphantom {b a}} 
\right. \kern-\nulldelimiterspace} a$ \textit{и} $a\lambda _2 \left( {M+S} 
\right)+\left( {\delta \beta +Sb} \right)\ge 0;$

\textit{3) }$\alpha ^2-4\beta >0,\;\;\lambda _2 <-b \mathord{\left/ {\vphantom {b a}} 
\right. \kern-\nulldelimiterspace} a$, $a\lambda _2 \left( {M+S} 
\right)+\left( {\delta \beta +Sb} \right)<0$ \textit{и}
\[
\left( {\frac{a\lambda _1 \left( {M+S} \right)+\left( {\delta \beta +Sb} 
\right)}{a\lambda _2 \left( {M+S} \right)+\left( {\delta \beta +Sb} 
\right)}} \right)^{\frac{\lambda _1 }{\lambda _2 -\lambda _1 }}\left( 
{a\lambda _1 \left( {M+S} \right)+\left( {\delta \beta +Sb} \right)} 
\right)>
\]
\begin{equation}
>\left( {Mb-\delta \beta } \right)-\left( {\frac{a\lambda _1 +b}{a\lambda _2 
+b}} \right)^{\frac{\lambda _1 }{\lambda _2 -\lambda _1 }}\left( {M-S} 
\right)\left( {a\lambda _1 +b} \right);
\end{equation}

\textit{4) }$ \alpha ^2-4\beta =0,\;\;\lambda _1 =\lambda _2 =\lambda >-b 
\mathord{\left/ {\vphantom {b a}} \right. \kern-\nulldelimiterspace} a;$

\textit{5) }$ \alpha ^2-4\beta =0,\;\;\lambda <-b \mathord{\left/ {\vphantom {b a}} 
\right. \kern-\nulldelimiterspace} a$ \textit{и} $a\lambda \left( {M+S} \right)+\left( 
{\delta \beta +Sb} \right)\ge 0;$

\textit{6) }$ \alpha ^2-4\beta =0,\;\;\lambda <-b \mathord{\left/ {\vphantom {b a}} 
\right. \kern-\nulldelimiterspace} a$, $a\lambda \left( {M+S} \right)+\left( 
{\delta \beta +Sb} \right)<0$ \textit{и}
\[

$ \textit{такая, что в системе (1) существует предельный цикл.}

\textit{Доказательство.} \textbf{1).} Рассмотрим случай $\alpha ^2-4\beta >0,\;\;\lambda _2 <-b 
\mathord{\left/ {\vphantom {b a}} \right. \kern-\nulldelimiterspace} a$, 
$a\lambda _2 \left( {M+S} \right)+\left( {\delta \beta +Sb} \right)<0$. 

Пусть неравенство (10) не выполнено. 

Предположим сначала, что верно равенство
\[
\left( {\frac{a\lambda _1 \left( {M+S} \right)+\left( {\delta \beta +Sb} 
\right)}{a\lambda _2 \left( {M+S} \right)+\left( {\delta \beta +Sb} 
\right)}} \right)^{\frac{\lambda _1 }{\lambda _2 -\lambda _1 }}\left( 
{a\lambda _1 \left( {M+S} \right)+\left( {\delta \beta +Sb} \right)} 
\right)=
\]
\begin{equation}
=\left( {Mb-\delta \beta } \right)+\left( {\frac{a\lambda _1 +b}{a\lambda _2 
+b}} \right)^{\frac{\lambda _1 }{\lambda _2 -\lambda _1 }}\left( {M-S} 
\right)\left( {a\lambda _1 +b} \right).
\end{equation}
Равенство (23) получено предельным переходом при $\varepsilon \to 0$ из 
равенства
\[
\left( {\frac{a\lambda _1 \left( {\bar {M}+\bar {S}} \right)+\left( {\delta 
\beta +\bar {S}b} \right)}{a\lambda _2 \left( {\bar {M}+\bar {S}} 
\right)+\left( {\delta \beta +\bar {S}b} \right)}} \right)^{\frac{\lambda _1 
}{\lambda _2 -\lambda _1 }}\left( {a\lambda _1 \left( {\bar {M}+\bar {S}} 
\right)+\left( {\delta \beta +\bar {S}b} \right)} \right)=
\]
\begin{equation}
=\left( {\bar {M}b-\delta \beta } \right)+\left( {\frac{a\lambda _1 
+b}{a\lambda _2 +b}} \right)^{\frac{\lambda _1 }{\lambda _2 -\lambda _1 
}}\left( {\bar {M}-\bar {S}} \right)\left( {a\lambda _1 +b} \right).
\end{equation}
Из рассуждений пункта \textbf{2 }доказательства теоремы 1 следует, что 
равенство (24) равносильно условию $ay_{\bar {M}} +bx_{\bar {M}} =-\delta $. 
И в этом случае траектория системы (6), проходящая через точку $\left( {\eta 
_{\bar {M}} ,\;\gamma _{\bar {M}} } \right)$, пересекает ось $Ox$ в точке 
$\left( {x_{\bar {S}} ,\;0} \right)$, а траектория системы (7), проходящая 
через $\left( {x_{\bar {S}} ,\;0} \right)$, достигает прямой $ay+bx=-\delta 
$ в точке $\left( {-\eta _{\bar {M}} ,\;-\gamma _{\bar {M}} } \right)$ (рис. 
5).

Указанные траектории систем (6) и (7), ``сшитые'' в точке $\left( {x_{\bar 
{S}} ,\;0} \right)$, вместе с симметричными им траекториями систем (8) и 
(9), проходящими через точку $\left( {-x_{\bar {S}} ,\;0} \right)$, образуют 
замкнутый контур.

Переходя к пределу при $\varepsilon \to 0$, получим: $\eta _{\bar {M}} \to 
\eta _M ,{\kern 1pt}\;\;\gamma _{\bar {M}} \to \gamma _M $, $t_{\bar {S}} 
\to t_S $, $x_{\bar {S}} \to x_S $. И если выполнено равенство (23), то 
траектория системы
\begin{equation}
\left\{ {\begin{array}{l}
 \dot {x}=y \\ 
 \dot {y}=-\alpha y-\beta x-S,\; \\ 
 \end{array}} \right.
\end{equation}
проходящая через точку $\left( {\eta _M ,\;\gamma _M } \right)$, пересекает 
ось $Ox$ в точке $\left( {x_S ,\;0} \right)$. 

Траектория системы
\begin{equation}
\left\{ {\begin{array}{l}
 \dot {x}=y \\ 
 \dot {y}=-\alpha y-\beta x-M, \\ 
 \end{array}} \right.
\end{equation}
проходящая через $\left( {x_S ,\;0} \right)$, достигает прямой 
$ay+bx=-\delta $ в точке $\left( {-\eta _M ,\;-\gamma _M } \right)$. 

Аналогично траектория системы
\begin{equation}
\left\{ {\begin{array}{l}
 \dot {x}=y \\ 
 \dot {y}=-\alpha y-\beta x+S,\; \\ 
 \end{array}} \right.
\end{equation}
проходящая через $\left( {-\eta _M ,\;-\gamma _M } \right)$, пересекает ось 
$Ox$ в точке $\left( {-x_S ,\;0} \right)$, и траектория системы
\begin{equation}
\left\{ {\begin{array}{l}
 \dot {x}=y \\ 
 \dot {y}=-\alpha y-\beta x+M,\; \\ 
 \end{array}} \right.
\end{equation}
проходящая через $\left( {-x_S ,\;0} \right)$, достигает прямой 
$ay+bx=\delta $ в точке $\left( {\eta _M ,\;\gamma _M } \right)$. 

Куски указанных траекторий систем (25)- (28) образуют замкнутый контур.

Введем функцию $\varphi \,\left[ {t,\sigma ,\varphi _0 } \right]$ следующим 
образом:
\begin{equation}
\varphi \,\left[ {t,\sigma ,\varphi _0 } \right]=\left\{ {\begin{array}{l}
 S,\quad \;\,\mbox{если}\quad t\in \left( {\tau _{4k} ,\;\tau _{4k+1} } 
\right); \\ 
 M,\quad \mbox{если}\quad t\in \left( {\tau _{4k+1} ,\;\tau _{4k+2} } 
\right); \\ 
 -S,\quad \mbox{если}\quad t\in \left( {\tau _{4k+2} ,\;\tau _{4k+3} } 
\right); \\ 
 -M,\;\;\mbox{если}\quad t\in \left( {\tau _{4k+3} ,\;\tau _{4k+4} } 
\right). \\ 
 \end{array}} \right.
\end{equation}
Определим моменты времени $\tau _k $: 

Пусть $\tau _0 =0$ и $\tau _1 =t_S $. При $t=\tau _0 $ решение системы (25) 
$\left( {x_S \left( t \right),\;y_S \left( t \right)} \right)$ проходит 
через точку $\left( {\eta _M ,\;\gamma _M } \right)$, а при $t=\tau _1 $ - 
через точку $\left( {x_S ,\;0} \right)$. 

Пусть при $t=\tau _1 $ решение системы (26) $\left( {x_M \left( t 
\right),\;y_M \left( t \right)} \right)$ проходит через точку $\left( {x_S 
,\;0} \right)$. Определим $\tau _2 $ как момент времени, при котором это 
решение попадает в точку с координатами $\left( {-\eta _M ,\;-\gamma _M } 
\right)$ на прямой $ay+bx=-\delta $.

Далее рассмотрим решение системы (27) $\left( {x_{-S} \left( t 
\right),\;y_{-S} \left( t \right)} \right)$, которое при $t=\tau _2 $ 
проходит через точку $\left( {-\eta _M ,\;-\gamma _M } \right)$. Определим 
$\tau _3 $ как момент времени, при котором это решение попадает в точку 
$\left( {-x_S ,\;0} \right)$ на оси $Ox$.

Пусть при $t=\tau _3 $ решение системы (28) $\left( {x_{-M} \left( t 
\right),\;y_{-M} \left( t \right)} \right)$ проходит через точку $\left( 
{-x_S ,\;0} \right)$. Тогда $\tau _4 $ определяется как момент времени, при 
котором это решение попадает в точку с координатами $\left( {\eta _M 
,\;\gamma _M } \right)$ на прямой $ay+bx=\delta $. 

Определяя далее числа $\tau _k $ аналогичным образом, получим, что при 
выполнении условия (23) в системе (1) с функцией $\varphi \,\left[ {t,\sigma 
,\varphi _0 } \right]$, заданной равенством (29), существует предельный 
цикл.

Пусть теперь выполнено неравенство 
\[
\left( {\frac{a\lambda _1 \left( {M+S} \right)+\left( {\delta \beta +Sb} 
\right)}{a\lambda _2 \left( {M+S} \right)+\left( {\delta \beta +Sb} 
\right)}} \right)^{\frac{\lambda _1 }{\lambda _2 -\lambda _1 }}\left( 
{a\lambda _1 \left( {M+S} \right)+\left( {\delta \beta +Sb} \right)} 
\right)<
\]
\begin{equation}
<\left( {Mb-\delta \beta } \right)+\left( {\frac{a\lambda _1 +b}{a\lambda _2 
+b}} \right)^{\frac{\lambda _1 }{\lambda _2 -\lambda _1 }}\left( {M-S} 
\right)\left( {a\lambda _1 +b} \right),
\end{equation}
которое получено предельным переходом при $\varepsilon \to 0$ из неравенства
\[
\left( {\frac{a\lambda _1 \left( {\bar {M}+\bar {S}} \right)+\left( {\delta 
\beta +\bar {S}b} \right)}{a\lambda _2 \left( {\bar {M}+\bar {S}} 
\right)+\left( {\delta \beta +\bar {S}b} \right)}} \right)^{\frac{\lambda _1 
}{\lambda _2 -\lambda _1 }}\left( {a\lambda _1 \left( {\bar {M}+\bar {S}} 
\right)+\left( {\delta \beta +\bar {S}b} \right)} \right)<
\]
\begin{equation}
<\left( {\bar {M}b-\delta \beta } \right)+\left( {\frac{a\lambda _1 
+b}{a\lambda _2 +b}} \right)^{\frac{\lambda _1 }{\lambda _2 -\lambda _1 
}}\left( {\bar {M}-\bar {S}} \right)\left( {a\lambda _1 +b} \right).
\end{equation}
Согласно пункту \textbf{2 }доказательства теоремы 1 неравенство (30) 
равносильно условию $ay_{\bar {M}} +bx_{\bar {M}} <-\delta $. И в этом 
случае траектория системы (6), проходящая через точку $\left( {\eta _{\bar 
{M}} ,\;\gamma _{\bar {M}} } \right)$, ``сшивается'' в точке $\left( 
{x_{\bar {S}} ,\;0} \right)$ с траекторией системы (7), которая достигает 
прямой $ay+bx=-\delta $ в некоторой точке $\left( {-x_{\bar {M}} ,\,-y{ 
}_{\bar {M}}} \right)$, где $-y_{\bar {M}} <-\gamma _{\bar {M}} $. 

В силу симметрии траектория системы (8), проходящая через точку $\left( 
{-\eta _{\bar {M}} ,\;-\gamma _{\bar {M}} } \right)$, ``сшивается'' в точке 
$\left( {-x_{\bar {S}} ,\;0} \right)$ с траекторией системы (9), которая 
достигает прямой $ay+bx=\delta $ в точке $\left( {x_{\bar {M}} ,\,\,y{ 
}_{\bar {M}}} \right)$.

В этом случае существует точка $\left( {\tilde {x},\,\,\tilde {y}} \right)$ 
($\gamma _{\bar {M}} <\tilde {y}<y_{\bar {M}} )$ на луче $L_2 $ такая, что 
траектория системы (6), проходящая через $\left( {\tilde {x},\,\,\tilde {y}} 
\right)$, ``сшивается'' на оси $Ox$ с траекторией системы (7), достигающей 
луча $L_1 $ в симметричной точке $\left( {-\tilde {x},\,\,-\tilde {y}} 
\right)$. Существование точки $\left( {\tilde {x},\,\,\tilde {y}} \right)$ 
доказывается с помощью построения отображения сжатия по траекториям систем 
(6)-(9) аналогично [5].

Указанные траектории (6) и (7), ``сшитые'' в некоторой точке $\left( {\tilde 
{x}_{\bar {S}} ,\;0} \right)$, вместе с симметричными им траекториями систем 
(8) и (9), ``сшитыми'' в точке $\left( {-\tilde {x}_{\bar {S}} ,\;0} 
\right)$, образуют замкнутый контур.

Далее переходя к пределу при $\varepsilon \to 0$ и определяя функцию 
$\varphi \,\left[ {t,\sigma ,\varphi _0 } \right]$ аналогично предыдущему 
случаю, получим, что в системе (1) с функцией $\varphi \,\left[ {t,\sigma 
,\varphi _0 } \right]$ существует предельный цикл, если выполнено 
неравенство (30).

В остальных случаях (если $\alpha ^2-4\beta =0,\;\;\lambda <-b 
\mathord{\left/ {\vphantom {b a}} \right. \kern-\nulldelimiterspace} a$, 
$a\lambda \left( {M+S} \right)+\left( {\delta \beta +Sb} \right)<0$ и не 
выполнено неравенство (11), или $\alpha ^2-4\beta <0,\;\;\lambda _{1,2} 
=v\pm iw$, и не выполнено неравенство (12)) доказательство проводится 
аналогично. 

Теорема 2 доказана.

\textbf{Теорема 3}. \textit{а). Если }$\delta \beta -Mb<0,$ \textit{то найдется функция }$\varphi \,\left[ {t,\sigma ,\varphi _0 } \right]$ \textit{класса } $K_{S,M}$ \textit{ такая, что в системе (1) существует предельный цикл.}

\textit{б). Если }$\delta \beta -Mb=0$\textit{, то найдется функция }$\varphi \,\left[ {t,\sigma ,\varphi _0 } \right]$ \textit{класса} $K_{S,M}$ \textit{ такая, что в системе (1) существует предельный цикл или замкнутый контур, ``сшитый'' из кусков траекторий и особых точек системы.}

Для доказательства теоремы 3 достаточно положить $\varphi _1 \left( 
{t,\sigma } \right)\equiv M$, ${\varphi _2 \left( {t,\sigma } \right)\equiv 
-M}$. Если $\delta \beta -Mb<0$, то в системе (1) существует предельный цикл, 
если $\delta \beta -Mb=0$, то существует предельный цикл или замкнутый 
контур, ``сшитый'' из кусков траекторий и особых точек системы [5].

\textbf{Теорема 4}. \textit{Пусть} $b\le 0$\textit{ и выполнено одно из следующих условий 1 - 5:}

\textit{1) }$\alpha ^2-4\beta >0$, $ a\lambda _2 \left( {M+S} \right)+\left( {\delta 
\beta +Sb} \right)\ge 0;$

\textit{2) }$\alpha ^2-4\beta >0$, $ a\lambda _2 \left( {M+S} \right)+\left( {\delta 
\beta +Sb} \right)<0$\textit{ и верно неравенство (10); }

\textit{3) }$\alpha ^2-4\beta =0$\textit{ и }$a\lambda \left( {M+S} \right)+\left( {\delta \beta 
+Sb} \right)\ge 0;$

\textit{4) }$\alpha ^2-4\beta =0$, $ a\lambda \left( {M+S} \right)+\left( {\delta \beta 
+Sb} \right)<0$\textit{ и верно неравенство (11); }

\textit{5) }$\alpha ^2-4\beta <0$\textit{ и верно неравенство (12).}

\textit{Тогда система (1) абсолютно устойчива в классе нелинейностей }$K_{S,M} .$

\textit{Если ни одно из условий 1 -- 5 не выполняется, то найдется функция }$\varphi \,\left[ {t,\sigma ,\varphi _0 } \right]$\textit{ из класса }$K_{S,M} $\textit{ такая, что в системе (1) существует предельный цикл.}

Доказательство теоремы 8 аналогично доказательству теорем 1, 2. В случае 
$b<0$ меняется наклон лучей $L_1 $ и $L_2 $, в случае $b=0$ лучи $L_1 $ и 
$L_2 $ параллельны оси $y=0$.
