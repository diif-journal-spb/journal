Abstract. This paper studies a modified Chua system with a discontinuous
sign(В· ) nonlinearity. Periodic regimes are localized using the harmonic bal-
ance method, Boiko locus, and event-driven numerical simulation by Piiroinen-
Kuznetsov method. A hidden stable limit cycle is found for one parameter set,
while a self-excited stable limit cycle is obtained for another.
