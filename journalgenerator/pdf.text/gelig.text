Abstract
In this paper we consider the system of di erential equations for which the
elements of ob ject matrix are functions of the system state and time. These
functions are perturbed by uncertain functionals.
Using the spectral decomposition of the ob ject matrix and the Lyapunov
quadratic function with unit matrix we obtain the su cient conditions of global
exponential stability of the system considered.
The similar system is studied in the case of scalar control, on the assumption
that the distribution vector depends on the state. Using the modal approach in
the condition of the uniform controllability we perform the synthesis of scalar
control which provides the global exponential stability of closed loop system.
Analogous results are obtained for discrete nonlinear uncertain systems with
the same structure.
Keywords: continuous nonlinear di erential uncertain systems, global
exponential stability , stabilization, discrete nonlinear uncertain systems
